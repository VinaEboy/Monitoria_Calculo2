# Repositório da Monitoria de Cálculo 2

Este repositório contém simulados, correções de provas e soluções de exercícios da disciplina de **Cálculo 2**, elaboradas pelo monitor para auxiliar no aprendizado e na compreensão dos conceitos abordados ao longo do curso. 

Também contém os projetos da Turma Honors de Cálculo 2 (2024/2), lecionado pelo professor Lucas Pedroso, que são implementações em Python de métodos númericos que usam conceito da matéria, como o Método do Gradiente para minimizar funções em R^N. 

- Caso encontre algum erro ou tenha dúvidas, sinta-se à vontade para entrar em contato ou sugerir melhorias. O meu contato está na página inicial do meu GitHub.
